## Aplicação de Cadernos e Registros

Projeto feito para o Bootcamp de C# e .NET da DIO oferecido pela Avanade.


Por enquanto, é possível fazer as seguintes ações:
 - Adicionar Caderno
 - Adicionar Registro
 - Listar cadernos
 - Listar registros
